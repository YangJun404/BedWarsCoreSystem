# 关于地图导入系统

# 地图辅助工具
选择器
编辑器
    识别符号：id，名称，微观模型，图标、简介
    基础信息：地图（资源点、边界{速度}、中心点、建筑团队），队伍（出生点与床）
    额外信息：特殊地图，不要求检测恢复，公开/私有化，绑定随机事件
辅助器
    基础框架：微观模型，地图模版
    信息辅助：当 debug 玩家到地图本体、地图复制体时自动介绍，地图主题，debug玩家可以参与到一张地图的制作团队之中、之后自动传送到地图本体，提供起床战争地图制作教程
    另类辅助：生成模版结构，替换周围白色方块的颜色至队伍颜色，快速传送到一个地图，添加传送点，合作平台，常用功能（高亮实体、高亮特殊方块、游戏规则快速更改、）

要求1：不复制地图，直接根据中心点计算资源点的相对坐标
要求2：期望有文字展示实体等可视化实体
